## Stack

- Next.js (Static Export Configured)
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server (Standard):
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
